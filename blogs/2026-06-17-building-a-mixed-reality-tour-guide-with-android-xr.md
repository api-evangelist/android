This walkthrough builds an XR tour guide app combining the Geospatial API (now in preview for ARCore Jetpack XR), the Gemini API with Firebase AI Logic, Google Maps Grounding, and the Jetpack XR SDK. It uses the Visual Positioning System for sub-meter location accuracy, Gemini for structured JSON tour itineraries, and the gemini-2.5-flash-tts model for dynamic voiceovers, with 3D rendering via Jetpack Compose for XR. The Android XR Developer Catalyst Program is now accepting applications for XREAL Project Aura devkits.
