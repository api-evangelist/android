Android developer verification protections take effect September 30, 2026, rolling out first to Brazil, Indonesia, Singapore, and Thailand across seven app stores including Google Play, Samsung, and Xiaomi. New APIs streamline registration: the Android Developer ID Status API checks package registration status and the Android Developer Console API enables bulk registration and CI/CD integration, both with OAuth delegation. Limited distribution accounts for students and hobbyists (sharing to 20 devices) require no government ID or fees.
