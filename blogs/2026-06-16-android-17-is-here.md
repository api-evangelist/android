Android 17 has launched on supported Pixel devices, positioning the platform as shifting "from an operating system to an intelligence system." Key features include AppFunctions expansion that lets apps expose capabilities as orchestratable tools for Android MCP and AI agents like Google Gemini, plus mandatory large-screen resizability reflecting the 580+ million large-screen devices in use. Source code is now available at AOSP.
